# CryptoConverter

Веб-приложение для отслеживания курсов криптовалют и конвертации. Курсы берутся с CoinGecko API

## Быстрый старт

'''
pip install -r requirements.txt
python app.py
В браузере: http://localhost:5000
'''

## Функционал

Главная - курсы BTC, USDT и выбранной крипты

Конвертер - обмен крипты на фиат и обратно

Настройки - выбор валюты, темы, частоты обновления
